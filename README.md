## 关于应用

「股息日历」是一款面向 A 股用户的分红与股息率查询工具。你可以：

- 按股票代码或名称搜索
- 查看历史分红、现价与股息率
- 管理多组合并按权重测算组合股息率

## 常见问题

### 1. 为什么搜不到某只股票？

请确认输入的是 A 股 6 位代码或常用简称。若网络异常，可稍后重试；仍无法搜索时，请通过下方邮箱反馈代码与截图。

### 2. 分红或行情数据不准怎么办？

应用展示的是公开渠道数据，可能存在延迟或口径差异。请以交易所、上市公司公告及券商客户端为准。若发现明显错误，欢迎邮件告知股票代码与问题页面。

### 3. 组合数据保存在哪里？

搜索历史与组合配置保存在你的**设备本地**，不会上传到我们的服务器。卸载应用或清除应用数据会导致本地记录丢失。

### 4. 股息率是如何计算的？

- **个股**：按近 1 年完整年度分红及近 2～5 年平均分红，结合现价估算股息率  
- **组合**：按你设置的权重加权  

最新年度若分红尚未实施完毕，可能包含预期补全，详情页会有标注。

### 5. 应用需要登录或付费吗？

当前版本无需登录，无内购。后续若功能调整，将在应用内或本页说明。

### 6. 如何反馈问题或建议？

请发送邮件至支持邮箱，尽量附上：

- 系统版本  
- 应用版本  
- 股票代码  
- 问题描述与截图  

便于我们尽快排查。
